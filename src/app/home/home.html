Opoint ZipFeed

Download latest RSS feed

This web page allows you to download your latest compiled and zipped RSS feed from the Opoint sources.

Right now, you can only download the archive corresponding to a single search profile ("PST html for feed", profile number 263596) that has been configured in your Opoint Portal. This feed will contain the latest 100 articles matching your profile.

The zipped archive will contain your feed XML (file named rss.xml) and all the compiled HTML files for intranet browsing.

Soon, you will start receiving PDF files with your feed too.

This feed delivery will only work if your intranet has a location named http://opoint.intranett.

Click to download Zip Feed

How to deploy the Zip Feed in your intranet

  1. Download and save the Zip archive on your machine
  2. Extract the contents of the Zip archive into the root of your intranet feed server (this is the folder where the address http://opoint.intranett points).
    Extracting the Zip archive will overwrite any already existing rss.xml file and it will also update the contents of the folder named files/, if necessary
  3. Make sure that your web server has the necessary permissions to read the deployed files
  4. Point your browser to http://opoint.intranett/rss.xml to view the latest deployed RSS feed
